# Roulette Game

## Опис
Це проста 2D-гра "Рулетка", яка використовує Phaser. Гравець натискає кнопку "Spin", щоб обертати колесо, після чого випадкове число від 0 до 36 показується на екрані.

## Інструкція
1. Встановіть залежності:
   ```bash
   npm install

2. Запустіть проект:
   
   npm run dev
